# Components List – FPV Racing Drone (5 Inch)

## Flight Controller
**SpeedyBee F405 V3**
- STM32F405 MCU  
- BMI270 gyro  
- Barometer  
- Blackbox (MicroSD)  
- 6 UARTs  
- Bluetooth configuration  
- Betaflight/INAV compatible  

## ESC
**50A BLHeli_32 4-in-1 ESC**
- Continuous: 50A  
- Burst: 55–60A  
- Firmware: BLHeli_32  
- Supports DShot300/600/1200  
- 4S–6S compatible  

## Motors
**T-Motor Velox V3 2207 2050KV**
- KV: 2050  
- Stator: 2207  
- Supports 4S–6S  
- Best with 5-inch tri-blade props  

## Propellers
**5-Inch Tri-Blade Props**
- Polycarbonate  
- High thrust and stable response  
- Standard pitch range for racing  

## Video Transmitter (VTX)
**AKK Race Ranger 1.6W**
- Power: 25 / 200 / 600 / 1600 mW  
- Input: 7–26V  
- SmartAudio support  
- Long-range, high-power VTX  

## Battery
**LiPo 6S 4000mAh 130C**
- 6S  
- 4000mAh  
- 130C discharge  
- Suitable for powerful 5-inch builds  

## Frame
**5-Inch Carbon Fiber Frame**
- 3K carbon fiber  
- 225–235mm wheelbase (typical)  
- 4mm arms  
- Supports 20×20 & 30×30 stack mounting
